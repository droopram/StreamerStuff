# Ninja's Slicing Alert

Legendary Twitch Streamer Ninja has a very unique alert where the username gets sliced in half. I recreated this alert in streamelements which supports twitch, youtube and facebook gaming.

It is pretty customizable! You can change the sprite, timing, colors etc.
